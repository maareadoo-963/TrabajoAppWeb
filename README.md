# TrabajoAppWeb
**Descripción del trabajo:** 
El proyecto consiste en una página web informativa para unos Talleres Formativos 2025 organizados por el Centro Cultural de la ciudad. La web presenta de forma clara y atractiva la información esencial sobre el evento: qué talleres se ofrecen, fechas, horarios, lugar de celebración y datos de contacto. El diseño está realizado con Bootstrap 5 mediante CDN, siguiendo una estructura estática pero visualmente ordenada. El objetivo es que los usuarios puedan consultar fácilmente la oferta formativa y los detalles prácticos del evento.
**Secciones usadas con su número de columnas de Bootstrap usadas**
El Navbar utiliza una sola fila para el menú horizontal. La sección Hero usa una columna centrada para el título principal, fechas y botones. La sección de Talleres está maquetada con una fila de tres columnas (col-4) que muestran tres tarjetas con los talleres disponibles. La sección de Información práctica utiliza dos columnas (col-6) para mostrar fechas en una y lugar en otra. La sección de Contacto emplea una columna centrada (col-6 mx-auto) con los datos de contacto. El Footer usa una columna simple para el copyright.
**Componentes prediseñados de Bootstrap usados**
Navbar (barra de navegación)
Buttons (botones con estilos btn-primary, btn-light, btn-outline-light)
Cards (tarjetas para mostrar los talleres)
Nav (menú de navegación simple)
**Descripción de los diferentes commits y mejoras implementadas de cada uno de ellos**
Commit 1: Creación del navbar y la sección de inicio. Se implementó el menú de navegación con enlaces a las diferentes secciones y la sección Hero con título, fechas del evento y botones de llamada a la acción.
Commit 2: Añadida la sección de talleres. Se maquetaron tres tarjetas con imágenes, títulos, descripciones y botones de información, distribuidas en tres columnas iguales.
Commit 3: Implementadas las secciones de información, contacto y footer. Se añadió la información práctica en dos columnas, los datos de contacto centrados y el pie de página con copyright. Se ajustaron espacios y detalles finales.
